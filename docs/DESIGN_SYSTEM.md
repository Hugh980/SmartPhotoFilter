# SmartPhotoFilter Design System

## Product Fit

SmartPhotoFilter is a desktop productivity tool for high-volume photo review. The UI
is optimized for scanning, comparing, adjusting thresholds, and exporting batches.

## Visual Tokens

| Role | Value |
|---|---|
| Primary | `#1E3A5F` |
| Secondary | `#2563EB` |
| Action | `#059669` |
| Background | `#F8FAFC` |
| Surface | `#FFFFFF` |
| Text | `#0F172A` |
| Muted text | `#64748B` |
| Border | `#E4E7EB` |
| Destructive | `#DC2626` |

## UI Rules

- Use a three-pane workspace: settings, photo grid, result summary.
- Keep controls at least 44px tall.
- Disable primary actions during analysis and show progress.
- Use result labels and table text in addition to colored borders.
- Use 8px radius for panels and tiles.
- Keep the interface light, professional, and information-dense.
